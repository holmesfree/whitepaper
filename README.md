# HOLMES Manifesto

> *"The quality of mercy is not strained..."*

The official whitepaper/manifesto for the HOLMES token movement.

## Building

Requires a LaTeX distribution (texlive or mactex):

```bash
./build.sh
```

Or manually:

```bash
pdflatex holmes-whitepaper.tex
pdflatex holmes-whitepaper.tex  # Run twice for TOC
```

## Output

The built PDF will be at `holmes-whitepaper.pdf`.

## Contents

This manifesto covers:

1. The case for mercy and clemency
2. A poetic appeal to President Trump and society
3. HOLMES token specifications and mechanics
4. The movement and how to participate
5. Community values and roadmap
